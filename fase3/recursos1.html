Aprende Sintaxis y Selectores CSS

Aprende Sintaxis y Selectores CSS

En este artículo mostramos cómo un estudiante puede aprender los conceptos básicos del sistema de bases de datos usando (W3school CSS) que fue construido como un sistema educativo de tutoría inteligente usando la herramienta de autoría llamada (ITSB). El material de aprendizaje contiene CSS y HTML. Dividimos el material en un grupo de lecciones para principiantes que combina el sistema relacional y lecciones en el proceso de aprendizaje. El estudiante puede aprender usando ejemplos de CSS y tipos de colores CSS. Además, el sistema de tutoría inteligente no solo admite lecciones; pero ejercicios de diferentes niveles de dificultad para cada lección. Cuando un estudiante termina con éxito el primer nivel de dificultad de una lección, se le permite pasar al siguiente nivel de dificultad de los ejercicios de la lección.

Alawar, MW y Abu-Naser, SS (2017). CSS-Tutor: Un sistema de tutoría inteligente para CSS y HTML.
